游戏介绍
主题：									
	佣兵团类型的卡牌，通过攻击，目的是消灭对面所有的卡牌								
									
玩法：									
	猜硬币正反面，获得先手权，通过掷骰子获得移动步数，移动卡牌进入可攻击敌方的范围内，卡牌生命值为0则卡牌从场上消失，直到某一方没有卡牌可用，则游戏结束
